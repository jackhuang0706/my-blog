關於這個blog的一些操作

網站架構&部署

  • 下載nodejs及git
  • 打開 windows Powershall(沒有Mac版本的，因為我不會用=+=)

安裝hexo

  • 執行 npm install -g hexo-cli(參見備註)
  • 用以下程式碼創建新的blog資料夾
    • hexo init my-blog
    • cd my-blog
    • npm install
  • 啟動伺服器 hexo server
  • 可以先到localhost:4000看看有沒有網站

安裝NexT 主題

  • 一樣的環境
  • git clone https://github.com/theme-next/hexo-theme-next themes/next
  • 可用ls 看my-blog裡面有哪些資料夾和檔案
  • 進入 /_config.yml 文件(code /_config.yml 應該會用vscode或其他IDE打開這份文件)
  • 裡面有一行 theme: landscape，把landscape改成next，如 theme: next
  • 啟動伺服器 hexo server

備註

如果執行npm命令時出現
npm : 因為這個系統上已停用指令碼執行，所以無法載入 C:\Program Files\nodejs\npm.ps1 檔案。
類似這樣的指令時，須以管理員身分先允許程式碼的執行。
以下為用管理員身分執行的操作

  • Get-ExecutionPolicy，沒意外會是 Restricted
  • Set-ExecutionPolicy RemoteSigned 修改成允許執行程式碼，如果出現提示則輸入 Y
  • 可以重新回到 windows powershall 使用者模式重新試看看囉

新增post

輸入 hexo new post 名稱，新的markdown文件就會生成，可以到 \source\_posts 查看，並 code 名稱 去編輯文件，可以將文件設定 tag 和 categories

刪除post

進入 \source\_posts，輸入 rm 要刪除的post名稱，操作完也要 hexo clean hexo generate hexo deploy
