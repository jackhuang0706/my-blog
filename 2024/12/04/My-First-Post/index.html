My First Post

Veröffentlicht am 2024-12-04

反正就順便紀錄一下這個blog的架設過程

  • 下載nodejs及git
  • 打開 windows Powershall

安裝hexo

  • 執行 npm install -g hexo-cli
  • 用以下程式碼創建新的blog資料夾
    • hexo init my-blog
    • cd my-blog
    • npm install
  • 啟動伺服器 hexo server
  • 可以先到localhost:4000看看有沒有網站

安裝NexT 主題

  • 一樣的環境
  • git clone https://github.com/theme-next/hexo-theme-next themes/next
  • 可用ls 看my-blog裡面有哪些資料夾和檔案
  • 進入 /_config.yml 文件(code /_config.yml 應該會用vscode或其他IDE打開這份文件)
  • 裡面有一行 theme: landscape，把landscape改成next，如 theme: next
  • 啟動伺服器 hexo server

部屬到github

  • 創建一個Github Repository，我是取叫 my-blog
  • 再打開 /_config.yml 文件，進行以下設定(要改的部分):
    • url: https://使用者自己的名稱.github.io/my-blog
    • deploy: 這邊要改:
      • type: git
      • repo: https://github.com/使用者自己的名稱/my-blog.git
      • branch: gh-pages
